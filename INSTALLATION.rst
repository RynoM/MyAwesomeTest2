============
Installation
============

At the command line::

    $ pip install myawesometest2lib

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv myawesometest2lib
    $ pip install myawesometest2lib

Or, if you are using pipenv::

    $ pipenv install myawesometest2lib

Or, if you are using pipx::

    $ pipx install myawesometest2lib
